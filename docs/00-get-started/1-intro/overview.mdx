# Get Started with Moralis

Welcome to Moralis! This guide will get you up and running with our
comprehensive Web3 development platform in minutes. Moralis provides powerful
APIs to access blockchain data across 30+ chains, making Web3 development as
easy as Web2.

## Quick Overview

Moralis offers four main categories of tools to accelerate your Web3
development:

1. **[Web3 Data APIs](#web3-data-apis)** - Access NFTs, tokens, transactions,
   and DeFi data
2. **[Streams API](#streams-api)** - Real-time blockchain event monitoring
3. **[RPC Nodes](#rpc-nodes)** - Reliable blockchain infrastructure
4. **[Authentication API](#authentication-api)** - Web3 wallet authentication

---

-   A Moralis account ([Sign up for free](https://admin.moralis.com/register))

## Web3 Data APIs

Our Web3 Data APIs provide high-level access to blockchain data without the
complexity of direct RPC calls. Get NFTs, tokens, prices, and transaction data
with simple REST API calls.

### Your First API Call

Let's fetch all NFTs owned by a wallet address:

<Tabs>
<TabItem value="curl" label="cURL">

```bash
curl -X 'GET' \
  'https://deep-index.moralis.io/api/v2.2/0x26fcbd3afebbe28d0a8684f790c48368d21665b5/nft?chain=eth&format=decimal&media_items=false' \
  -H 'accept: application/json' \
  -H 'X-API-Key: YOUR_API_KEY'
```

</TabItem>
<TabItem value="javascript" label="JavaScript">

```javascript
const Moralis = require("moralis").default;

const runApp = async () => {
    await Moralis.start({
        apiKey: "YOUR_API_KEY",
    });

    const address = "0x26fcbd3afebbe28d0a8684f790c48368d21665b5";

    const response = await Moralis.EvmApi.nft.getWalletNFTs({
        chain: "0x1",
        format: "decimal",
        mediaItems: false,
        address: address,
    });

    console.log(response.raw);
};

runApp();
```

</TabItem>
<TabItem value="python" label="Python">

```python
from moralis import evm_api

api_key = "YOUR_API_KEY"

params = {
    "address": "0x26fcbd3afebbe28d0a8684f790c48368d21665b5",
    "chain": "eth",
    "format": "decimal",
    "media_items": False
}

result = evm_api.nft.get_wallet_nfts(
    api_key=api_key,
    params=params,
)

print(result)
```

</TabItem>
</Tabs>

## Streams API

Monitor blockchain events in real-time with our Streams API. Get instant
notifications when specific transactions, transfers, or smart contract events
occur.

### Create Your First Stream

Here's how to monitor all USDC transfers on Ethereum:

<Tabs>
<TabItem value="javascript" label="JavaScript">

```javascript
const Moralis = require("moralis").default;

const runApp = async () => {
    await Moralis.start({
        apiKey: "YOUR_API_KEY",
    });

    const stream = {
        chains: ["0x1"], // Ethereum
        description: "Monitor USDC transfers",
        tag: "USDC_transfers",
        webhookUrl: "https://YOUR_WEBHOOK_URL.com/webhook",
        abi: [
            {
                anonymous: false,
                inputs: [
                    { indexed: true, name: "from", type: "address" },
                    { indexed: true, name: "to", type: "address" },
                    { indexed: false, name: "value", type: "uint256" },
                ],
                name: "Transfer",
                type: "event",
            },
        ],
        advancedOptions: [
            {
                topic0: "Transfer(address,address,uint256)",
                filter: {
                    and: [
                        {
                            eq: [
                                "from",
                                "0xa0b86a33e6776c8b2a4d1a5b9b3f4e4f7f8e5f4f",
                            ],
                        },
                        { gt: ["value", 1000000] }, // Greater than $1 (6 decimals)
                    ],
                },
                includeContractLogs: true,
                includeInternalTxs: false,
            },
        ],
        includeContractLogs: true,
        includeInternalTxs: false,
    };

    const newStream = await Moralis.Streams.add(stream);
    console.log(newStream);
};

runApp();
```

</TabItem>

## RPC Nodes

Access reliable, high-performance RPC endpoints for direct blockchain
interaction. Perfect for sending transactions, deploying contracts, and custom
blockchain queries.

### Quick Connection

<Tabs>
<TabItem value="javascript" label="JavaScript (Web3.js)">

```javascript
const Web3 = require("web3");

// Replace with your Moralis RPC URL
const web3 = new Web3(
    "https://site1.moralis-nodes.com/eth/mainnet/YOUR_API_KEY"
);

// Get latest block
const latestBlock = await web3.eth.getBlockNumber();
console.log("Latest block:", latestBlock);

// Get account balance
const balance = await web3.eth.getBalance(
    "0x742d35cc659c5cfb5e751e53d5b6e6b3bd7c8b2a"
);
console.log("Balance:", web3.utils.fromWei(balance, "ether"), "ETH");
```

</TabItem>
<TabItem value="ethers" label="JavaScript (Ethers.js)">

```javascript
const { ethers } = require("ethers");

// Connect to Moralis RPC
const provider = new ethers.providers.JsonRpcProvider(
    "https://site1.moralis-nodes.com/eth/mainnet/YOUR_API_KEY"
);

// Get network info
const network = await provider.getNetwork();
console.log("Connected to:", network.name);

// Get account balance
const balance = await provider.getBalance(
    "0x742d35cc659c5cfb5e751e53d5b6e6b3bd7c8b2a"
);
console.log("Balance:", ethers.utils.formatEther(balance), "ETH");
```

## Authentication API

Implement secure Web3 wallet authentication with just a few lines of code.
Support for MetaMask, WalletConnect, and other popular wallets.

### Web3 Login Flow

<Tabs>
<TabItem value="javascript" label="JavaScript">

```javascript
const Moralis = require("moralis").default;

const runApp = async () => {
    await Moralis.start({
        apiKey: "YOUR_API_KEY",
    });

    // Request message for signing
    const { message } = await Moralis.Auth.requestMessage({
        address: "0x742d35cc659c5cfb5e751e53d5b6e6b3bd7c8b2a",
        chain: "0x1",
        networkType: "evm",
        domain: "moralis.io",
        statement: "Please sign this message to confirm your identity.",
        uri: "https://moralis.io/",
        expirationTime: "2023-01-01T00:00:00.000Z",
        timeout: 15,
    });

    console.log("Message to sign:", message);

    // After user signs the message with their wallet
    const signature = "USER_SIGNATURE_FROM_WALLET";

    // Verify the signature
    const { profileId } = await Moralis.Auth.verify({
        message,
        signature,
        networkType: "evm",
    });

    console.log("Verified user profile:", profileId);
};

runApp();
```

</TabItem>
<TabItem value="react" label="React Example">

```jsx
import { useMoralis } from "react-moralis";

function WalletConnect() {
    const { authenticate, isAuthenticated, user, logout } = useMoralis();

    const handleConnect = async () => {
        await authenticate({
            signingMessage: "Sign to verify your identity",
            chain: "eth",
        });
    };

    if (isAuthenticated) {
        return (
            <div>
                <h3>Welcome {user.get("ethAddress")}</h3>
                <button onClick={logout}>Disconnect</button>
            </div>
        );
    }

    return <button onClick={handleConnect}>Connect Wallet</button>;
}
```

</Tabs>
